## Mission

Improve **clive** — an LLM that drives a real shell/CLI through tmux — so it
completes real tasks more reliably and efficiently. Research focus: agents that
use a command line to work and communicate (tool-use, terminal/computer-use,
multi-agent CLI coordination). Optimise only clive's `open` config (system prompt,
toolset, observation/recovery policy, skills); never the frozen safety block.
